Map va Set haqida tushuncha

Map nima?

JavaScriptda Map ma'lumot tuzilmasi bo'lib, u kalit-qiymat juftlarini saqlash imkonini beradi. Map-dagi kalitlar takrorlanmas va har qanday ma'lumot turida bo'lishi mumkin.

Map misoli: 
const myMap = new Map();
myMap.set('kalit1', 'qiymat1');
myMap.set('kalit2', 'qiymat2');
console.log(myMap.get('kalit1')); // Natija: qiymat1

Set nima?

JavaScriptda Set takroriy qiymatlarni saqlamaydigan kolleksiya hisoblanadi. Bu noyob qiymatlarni saqlash uchun juda qulay vositadir.

Set misoli: 
const mySet = new Set();
mySet.add(1);
mySet.add(2);
mySet.add(2); // Takroriy qiymat e'tiborga olinmaydi
console.log(mySet); // Natija: Set { 1, 2 }

Map va Setning farqlari

  • Maqsad: Map kalit-qiymat juftliklarini boshqaradi, Set esa noyob qiymatlarni saqlaydi.
  • Tuzilishi: Map kalit va qiymatlardan tashkil topgan, Set faqat qiymatlardan iborat.
  • Metodlar: Mapda set, get, has; Setda add, delete, has.
